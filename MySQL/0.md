# 初识MySQL





## 1.数据库分类



#### 关系型数据库：

* MySQL，Oracle，Sql Server， SQLlite

* 通过表和表之间，行和列之间的关系进行数据存储

    

#### 非关系型数据库：

* Redis， MongoDB
* 非关系型数据库，对象存储，通过对象的自身属性决定



### __DBMS（数据库管理系统）__

* 数据库的管理软件

* MySQL，数据库管理系统
